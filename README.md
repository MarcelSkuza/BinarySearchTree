# Binary Search Tree
C ++ implementation of binary search tree algorithm. Uses simple word-counting program as an example implementation. 
## Makefile
 **Build**/
`nmake library`/
`nmake compile_test`/
`nmake wordcouter_compile`/
/
**Run tests**/
`nmake test`/
/
**Run word-counter**/
`nmake single_word`/
`nmake sentences`/
/
**Clean built files**/
`nmake clean`/
